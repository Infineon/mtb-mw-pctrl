MPPT (Maximum Power Point Tracking)

General Description

Context structure and sub-structures for the MPPT block.


The context structure contains all the necessary data for MPPT operation. It is divided into two main parts:

  • Configuration structure (mppt_config_t): Must be set by the application
  • Runtime variables structure (mppt_static_t): Managed automatically by the library The application must allocate the context structure (mppt_context_t) in a memory section of choice. It must set the values of the configuration sub-structure (mppt_context_t::config). The static sub-structure (mppt_context_t::vars) contains the static memory the block needs to operate. It must not be initialized by the application. The application has to call the block reset function (MpptReset()) to clear the static sub-structure and the block init function (MpptInit()) to initialize the internal configuration.
